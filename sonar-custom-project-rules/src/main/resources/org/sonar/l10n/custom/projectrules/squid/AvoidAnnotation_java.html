This rule detects usage of configured annotation

Noncompliant Code Example

class AvoidAnnotationCheck {

  int aField;

  @MyAnnotation
  public void aMethod() {

  }

  @Test // Noncompliant {{Avoid using annotation @Test}}
  public void aMethod() {

  }

}

Compliant Solution

class AvoidAnnotationCheck {

  int aField;

  @MyAnnotation
  public void aMethod() {

  } 

}
